Heterogeneous warehouse fleets typically include robots with different roles (e.g., carrier vehicles and loading agents) that must coordinate their movement, timing, and energy usage to sustain high throughput.
Designing controllers for such multi-robot systems is challenging due to large discrete action spaces, partial observability, sparse rewards, and safety constraints such as battery depletion.

This thesis investigates whether off-the-shelf \glspl{llm} can be used as high-level decision-makers for heterogeneous robot collaboration in a task-assignment warehouse environment with battery dynamics.
The approach converts the current multi-agent state into a compact natural-language description, queries an \gls{llm} to choose an action for each agent, and enforces feasibility by validating proposed actions against environment-defined valid action masks.
Different model sizes, planning architectures, and prompt designs are compared in the Battery-TA-RWARE environment.

The current work contributes (i) an end-to-end evaluation pipeline for \gls{llm}-driven multi-agent control, (ii) prompt and validation patterns tailored to discrete warehouse actions, and (iii) preliminary experimental results showing that prompt structure, planning architecture, and safeguard logic strongly influence whether \gls{llm}-based control remains productive under changing objectives.
